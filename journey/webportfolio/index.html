The creation of my portfolio has been long overdue. I have made several attempts over the past few years, and for a variety of reasons, it’s taken me this long. The current iteration you’re seeing right now uses a Hugo (the static site generator) and a theme I designed (I call it DEZERT). Making a static site from scratch took a lot more work than I expected, probably because I was learning as I went.

If my Hugo site was from scratch, then my previous iteration would have been from atoms. I built an entire web app, backend and everything, content management included. I utlized Django REST Framework and ReactJS. It had a lot of features, like an admin panel, a customized WYSIWYG editor using Slate.js (which was a joy to use), drafting support, and autosave. At some point I might write some articles since there were some interesting backend classes I wrote.
